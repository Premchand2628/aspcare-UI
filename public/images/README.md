# Image Assets

Place your image files in this directory.

## Required Images:

### Icons & UI Elements
- `user-avatar.png` - User profile picture
- `rewards-icon.png` - Rewards/trophy icon
- `notification-icon.png` - Notification bell icon
- `checkout-icon.png` - Checkout service icon
- `membership-icon.png` - Membership benefits icon
- `trophy.png` - Trophy/rewards image
- `orders.png` - Orders icon

### Car Images
- `car-main.png` - Main showcase car (white sports car with orange details)
- `car-wash.png` - Car being washed
- `car-detail.png` - Car detail view
- `car-wash-splash.png` - Car with water splash effect

## Image Specifications:

- **Icons**: 100x100px to 200x200px, PNG format with transparency
- **Car Images**: 800x600px or higher, PNG/JPG format
- **Profile Avatar**: 200x200px, PNG format with transparency

## Placeholder Images:

Currently, the app uses placeholder backgrounds. Replace these with actual images for production.

You can use:
- Free stock photos from Unsplash, Pexels, or Pixabay
- Custom illustrations
- Icon libraries like Font Awesome, Material Icons
